# Iris Flower Classification
![image_alt](https://github.com/shubhlovesdata/Unified_Mentor_Internship_Projects/blob/16f69f78eec020e79e4d3ca06879ade4d743ade5/Iris%20Classification/Iris-Dataset-Classification-1024x367.png)

This project focuses on building a machine learning model to classify iris flowers into three species ‚Äî **Setosa**, **Versicolor**, and **Virginica** ‚Äî based on their sepal and petal measurements.  
It is a classic supervised learning problem used to demonstrate classification techniques and model evaluation.

## Objective
- Understand the Iris dataset and its features
- Perform exploratory data analysis (EDA)
- Build and train classification models
- Evaluate model performance using standard metrics

## Dataset
- Source: dataset from Kaggle(https://www.kaggle.com/datasets/uciml/iris)
- Records: 150 samples
- Features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
- Target:
  - Iris-setosa
  - Iris-versicolor
  - Iris-virginica

## Exploratory Data Analysis
- Distribution analysis of sepal and petal features
- Relationship between features and flower species
- Identification of feature importance in classification

## ‚öôÔ∏è Machine Learning Models Used
The following models were implemented and evaluated:

- K-Nearest Neighbors (KNN)

- Accuracy Score

## üîç Result Analysis
- All models performed well with **high accuracy**, indicating the dataset is well-structured and separable.
- **Support Vector Machine (SVM)** achieved the **highest accuracy (98.33%)**, making it the best-performing model for this project.
- Logistic Regression and KNN also showed strong performance, confirming the reliability of the feature set.
- Decision Tree performed slightly lower but still achieved good accuracy.
